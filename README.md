# Renens Diaries

Envoie chaque jour à 20h un petit message sur le topic ntfy `Renensdiaries` :

- Bravo Dudu la journée est finie !
- Coucher de soleil à Renens
- Actus sport en Suisse (placeholder)
- Horoscope Verseau (placeholder)
- Petit texte "IA"
- Emoji d’animal mignon

Le script est dans `main.py` et utilise ntfy (`https://ntfy.sh/Renensdiaries`).
